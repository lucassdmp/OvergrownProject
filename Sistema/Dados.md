# Dados

## Introdução
Os dados são a representação numérica das capacidades dos personagens no sistema de RPG Overgrown. Eles são derivados dos atributos e influenciam diretamente as ações e resultados das jogadas.

## Estrutura dos Dados

Os dados são compostos por três elementos principais:

1. **Atributos**: Os valores base que definem as capacidades do personagem.
2. **Modificadores**: Bônus ou penalidades aplicados aos atributos em situações específicas.
3. **Resultados**: Os valores finais utilizados nas jogadas e interações do jogo.

## Rolamentos
As jogadas no sistema de Overgrown são realizadas utilizando dados de 20 lados (d20). O resultado do rolamento é somado ao valor do atributo relevante e quaisquer modificadores aplicáveis para determinar o sucesso ou falha da ação.

A cada 5 de atributos o personagem recebe um dado adicional para rolar, por exemplo:
- Atributo 1-4: 1d20
- Atributo 5-9: 2d20
- Atributo 10-14: 3d20
- Atributo 15-19: 4d20
- Atributo 20+: 5d20

Se limitando até 5 dados por rolamento, mas podendo ser aumentado por habilidades especiais, condições especiais ou passivas da árvore de talentos.
